= nanite

nanite is a self assembling slice building agent framework.

== GOALS:

1. Manage required package(unix/gems/whatever)
2. Able to bring a slice into a consistent state based on a manifest
3. Unit testable server configuration, run tests to get current state of slice
   see if it matches spec/manifest notify on diversion
4. Keep all files versioned in git
